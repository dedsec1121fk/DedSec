### 1. Install Termux
- Download the F-Droid app store from its official website.
- Open the F-Droid app and search for **Termux**.
- Install **Termux** from the F-Droid store.

### 2. Grant Storage Access to Termux
- Open the **Termux** app.
- Run the command:
  ```bash
  termux-setup-storage
  ```
- This command will allow **Termux** to access your device's storage.

### 3. Update Termux
- In the **Termux** terminal, run the command:
  ```bash
  pkg update && pkg upgrade
  ```
- This ensures that all packages are up to date.

### 4. Install Python
- Type the following command to install Python:
  ```bash
  pkg install python
  ```

### 5. Install Required Libraries
- Install necessary libraries such as `requests`, `phonenumbers`, and `googlesearch-python`:
  ```bash
  pip install requests phonenumbers googlesearch-python
  ```

### 6. Create a Folder for the Project
- Create a new directory named **OSINTDS** in your home directory:
  ```bash
  mkdir ~/OSINTDS
  ```
- Change to that directory with:
  ```bash
  cd ~/OSINTDS
  ```

### 7. Create the Python Script
- Use a text editor (like `nano`) to create a new file called `osintds.py`:
  ```bash
  nano osintds.py
  ```
- Paste the provided Python code into the editor.
- Save the file and exit the text editor by pressing `CTRL + X`, then `Y`, and `ENTER`.

### 8. Run the Program
- Start the program by typing:
  ```bash
  python osintds.py
  ```
- The program will display a menu with options to select from different OSINT tools.

### 9. Using the OSINTDS Tools
- **Email Lookup**: Enter an email address to check if it is valid and compromised.
- **Instagram Lookup**: Enter an Instagram username to retrieve public data about the user.
- **IP Lookup**: Enter an IP address to get geolocation information.
- **Phone Lookup**: Enter a phone number to validate it and get information about its carrier and region.
- **Web Search**: Enter a search query to perform a web search.
- **Username Lookup**: Enter a username to check if it exists on various social media platforms.

### 10. Stop the Program
- To stop the program, press `CTRL + C` in the **Termux** terminal.
