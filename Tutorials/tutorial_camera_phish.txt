1. **Install Termux:**
   - Download and install the F-Droid app store from its official website.
   - Open the F-Droid app and search for Termux.
   - Install Termux from the F-Droid store.

2. **Grant Storage Access to Termux:**
   - Open the Termux app.
   - Run the command `termux-setup-storage` to allow Termux to access your device's storage.

3. **Update Termux:**
   - In the Termux terminal, run the command `pkg update && pkg upgrade` to ensure all packages are up to date.

4. **Install Python:**
   - Type `pkg install python` to install Python in your Termux environment.

5. **Install Flask:**
   - After Python is installed, run `pip install flask` to install the Flask web framework.

6. **Install OpenSSH:**
   - Use the command `pkg install openssh` to install OpenSSH, which is needed for creating an SSH tunnel with Serveo.

7. **Create a Folder for the Project:**
   - Type `mkdir ~/Camera-Phish` to create a new directory named Camera-Phish in your home directory.
   - Change to that directory with `cd ~/Camera-Phish`.

8. **Create the Python Script:**
   - Use a text editor (like nano) to create a new file called `camera_phish.py`.
   - Open the text editor by typing `nano camera_phish.py`.
   - Paste the provided Python code into the editor.
   - Save the file and exit the text editor.

9. **Run the Program:**
   - Start the program by typing `python camera_phish.py`.
   - The program will start a Flask server and create a Serveo tunnel, displaying a URL in the terminal. This URL will look something like `https://<your-tunnel>.serveo.net`.

10. **Access the Camera:**
    - Open a web browser on your device or another device connected to the internet.
    - Enter the Serveo URL from the terminal into the browserâ€™s address bar.
    - The camera interface will open, and the camera will start capturing images automatically every 1.5 seconds.

11. **Where the Photos Are Saved:**
    - The captured photos will be saved in the folder `Camera-Phish`, which is located in the `Downloads` directory of your internal storage.
    - Specifically, the path to the saved images is `~/storage/downloads/Camera-Phish`. You can access this folder through any file manager app on your device.

12. **Stop the Program:**
    - To stop the program, press `CTRL + C` in the Termux terminal.
