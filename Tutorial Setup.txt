Download TermuxTermux: API and Termux: Styling from F-Droid.
Open Termux and then copy the entire command text that is below and pastein your Termux terminal,press y to everything it asks.
Then write:
git clone https://github.com/dedsec1121fk/DedSec

And then write:
python main_menu.py

Now everytime you run Termux you will can choose any tool you want easily.
